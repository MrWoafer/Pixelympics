# Pixelympics

A local multiplayer (plus CPU opponents) game comprised of 20+ Olympics minigames.

Written in C# with Unity.

# Events

* 100m, 200m, 400m
* Hurdles
* Javelin
* Hammer
* Shot Put
* Long Jump
* Triple Jump
* High Jump
* Pole Vault
* Rowing
* 100m Freestyle
* Archery
* Skeet
* Speed Climbing
* Ski Jump
* Karate
* Wrestling
* Weightlifting
